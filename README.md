# Netweb DCF Valuation Model

A premium, interactive Discounted Cash Flow (DCF) valuation dashboard for **Netweb Technologies India Ltd (NSE: NETWEB)**. This tool provides a professional-grade financial analysis, allowing users to project future cash flows and determine the intrinsic value of the company.

## 🚀 Project Overview

Netweb Technologies is a leading high-end computing solutions (HCS) provider in India. This application builds a 10-year Free Cash Flow to Firm (FCFF) model to estimate the company's valuation based on various financial assumptions.

### Key Features
- **Interactive Dashboard**: Real-time updates of intrinsic value based on user inputs.
- **Dynamic Assumptions**: Editable fields for revenue growth, EBITDA margins, WACC, and terminal growth.
- **Financial Projections**: 10-year detailed breakdown of NOPAT, CapEx, ΔNWC, and FCFF.
- **Sensitivity Analysis**: Multi-dimensional grid visualizing valuation across different WACC and Growth scenarios.
- **Investment Thesis**: Summary of bull and bear case drivers for the company.

## 🛠 Tech Stack
- **Framework**: React 19
- **Build Tool**: Vite
- **Styling**: Vanilla CSS (Inline)
- **Design**: Premium dark-themed dashboard with responsive layouts.

Follow these steps to run the project locally:

### 1. Prerequisites
Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

The project is already configured with essential dependencies in `package.json`. Navigate to the project directory:

```bash

```

Start the local development server:
```bash
npm run dev
```

Open your browser and navigate to:
[http://localhost:5173/](http://localhost:5173/)
